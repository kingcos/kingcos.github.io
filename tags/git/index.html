Zen of Git

2018-06-13
#Tips  #Git  #DevOps 
在软件开发的团队协作中，一个稳定的版本控制系统（Version Control System）是必不可少的，而 Git 已成为绝大多数开发团队中的首选。本文将简单谈谈 Git 和其相关的工作流、工具链，以及背后的思考，当然对于团队的习惯以及开发性质，其选择的侧重会略有不同，也欢迎大家互相交流。
