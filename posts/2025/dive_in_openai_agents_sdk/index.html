浅析 OpenAI Agents SDK

2025.03.12 by kingcos

概览

继 Manus 以及一众开源智能体框架之后，OpenAI 也推出了自己的智能体框架：OpenAI Agents SDK。本文将基于源码进行简单分析，窥探其实现原理与流程。

基本信息

  • 代码版本：c8f3cdd
  • 主要语言：Python
  • 开源协议：MIT

系统架构

在 OpenAI Agents SDK 中，官方定义了以下几个核心概念：

  1. Agent (智能体)：配置了指令、工具、护栏和移交功能的 LLM (大语言模型)
  2. Handoff (移交)：允许 Agent 将控制权转移给其他 Agent 以执行特定任务
  3. Guardrail (护栏)：用于输入和输出验证的可配置安全检查
  4. Tracing (追踪)：内置的 Agent 运行跟踪功能，可查看、调试和优化工作流

核心组件

Runner

OpenAI Agents SDK 的主入口是 Runner 类，其主要执行流程为：

  1. 创建 Agent，传入名称、指令（instructions，即 Agent 的系统提示词），也可以内置工具
  2. 执行 Runner（Runner.run）
  3. 调用 _run_single_turn 方法
  4. 执行 OpenAIResponsesModel 的 get_response -> _fetch_response
  5. 调用 LLM 的 create 方法
  6. 调用 _get_single_step_result_from_response 方法解析 LLM 的返回结果，并返回单步结果
  7. 直到达到最大轮数（max_turns）或 Agent 已经生成最终输出
