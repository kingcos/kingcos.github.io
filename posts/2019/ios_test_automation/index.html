有时通过网络连接会受很多限制或速度较慢，我们也可以将电脑本机端口映射到 USB 连接的手机端口：

brew install usbmuxd

iproxy 8100 8100

此时通过 http://localhost:8100/status 即可测试端口是否映射成功：

Appium

Appium 可以使用 npm 来安装，而 npm 可以使用 Homebrew 来安装（如果 npm 速度过慢，可以使用 cnpm）：

brew install npm

npm install appium -g
