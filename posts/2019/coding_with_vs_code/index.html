与 Visual Studio Code 共舞

2019-08-30 — 1 min read
#Tips  #🚧 

Preface

VS Code（全称 Visual Studio Code），是微软推出的一款插件化编辑器。比 IDE 轻量，又比终端体验更好，本文将罗列一些我在使用的设置、配置、插件等，也欢迎大家建言献策～

Debug

Settings

Name Effect
Auto Save 自动保存，防止意外
Format On Save 保存时自动格式化

Go

  • 插件：ms-vscode.Go
// launch.json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Launch",
      "type": "go",
      "request": "launch",
      "mode": "auto",
      "program": "${fileDirname}",
      "env": {
        // 注意以下环境变量的设置
        "GOPATH": "/usr/local/Cellar/go/1.12.9",
        "GOROOT": "/usr/local/Cellar/go/1.12.9/libexec",
        "GOBIN": "/usr/local/Cellar/go/1.12.9/bin"
      }
    }
  ]
}

// settings.json
{
  // 必要时可强制将环境变量改写
  "go.gopath": "/usr/local/Cellar/go/1.12.9/"
}

可在 VS Code 的 Terminal 输入 go env 查看当前的环境变量。

Ruby

  • 插件：rebornix.Ruby
// launch.json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Debug Local File",
      "type": "Ruby",
      "request": "launch",
      "program": "${workspaceRoot}/${fileBasename}"
    }
  ]
}

Gitlens

  • eamodio.gitlens
  • 方便查看 Git Blame。
