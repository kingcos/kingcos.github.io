iDay-11.15

2015-11-16
#Past 

今（昨）天又是一个看似忙碌其实无为的一天。

翻译到现在还是没填坑。关注人数增加的的确缓慢了很多。

双十一买的键盘退了，买了别人的二手的罗技键盘。发现自己买了好多二手的，因为便宜的价格而入手，也算是我现在没有现金收入的一种节流补偿手段吧。这种金钱观可能还会持续一段时间吧，毕竟我也是个纠结的人。

说说最近的精神状态吧，这一周基本上都没早起过了，不管我自己定多少闹钟，用不用 Sleep Cycle，都起不来，可能是晚上睡的太晚了吧，但是我又不喜欢早睡，因为只有晚上的时间是属于我一个人的，早已习惯了晚上一个人的世界，大家都睡了，安静，有效率。

双十一买的羽绒服到了，优衣库家的，说实话不是因为除了那次的事情才知道优衣库，而是洛阳有了实体店。在优衣库买衣服没有服务员跟着催你买，这样的感觉真好，有些影响就是，在那次事件以后，去试衣间还要带号，还不能把衣服穿出来让同行的人看看效果。不过也能理解。

这周还买了 Dash 这款 iOS 软件，不过我又申请退款了，对开发者有一定帮助，但是排版太差了。我个人不太喜欢。Apple 的软件退款体验也还好。

好啦，又到了计划时间，周一：

  1. Java 作业
  2. 毛概演讲
  3. 理发
  4. 翻译…

哦对了，顺便说一下 NY 意思是”南阳”，不是 New York…
