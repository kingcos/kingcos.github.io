Why English?

As we know, all the commands in Linux consist of the English names. So it’s easy to remember the commands with the help of English.

Basic Knowledge

Linux’s feature: Open source

Everybody can get, spread and even modify the source code freely

Linux plays an important role in Server-Side and Embeded platform

Linux has 4 major parts: Kernel Shell File-System and Applications

Linux has 2 kinds of versions: Kenel version and Distribution

Kernel has 3 numbers：r.x.y

1. r：The main kernel version

2. x：Even number for stable, while the odd number for beta

3. y：Count of modification

Common distribution：Mandriva Red Hat SUSE Debian Ubuntu Gentoo Slackware Red Flag

We don’t use the root account to login the system because it’s not safe

Fedora’s default graphical interface: GNOME

Fedora’s explorer: Mozilla Firefox

User prompt：Super user# common user$

Linux system running level is a kind of state.

3. Last line mode

Switch among 3 modes

2 -> 1 Press ‘ESC’

1 -> 3 Press ‘:’ and input ‘wq’ to save and exit

All the directories and devices in Linux exist by file

Linux file type

1. common file

2. directory file

/etc : Include most parts of configuration files in Linux system

Line Describtion
1 1:file type 2-4:file owner’s access permission 5-7:user group’s access permission 8-10:other user’s access permission

Linux Common Commands

cat […] [filename]

concatenate and print files

…:

  1. -n: Number the output lines, starting at 1.
  2. -b: Number the non-blank output lines, starting at 1.
  3. -s: Squeeze multiple adjacent empty lines, causing the output to be single spaced.
  4. -E: Output $ when each line’s end

sort […] [file]

sort lines of text files

…:

  1. -m
  2. -c
  3. -u
