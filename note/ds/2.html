指针

笔记：

使用指针可以对复杂数据进行处理，能对计算机的内存进行分配控制，在函数调用中使用指针还可以返回多个值

基本运算：

  • & 取地址，* 间接访问
  • 指针同整数进行加减操作：p + i: 从 p 位置开始的第 i 个该类型变量的地址
  • 同类型指针的相减操作：表示两个指针之间相隔的变量个数
  • 两个同类型指针可以使用关系运算符比较大小

指针与数组

  • C 语言中，数组名本身就是数组中第一个元素（下标为 0）的地址
  • 访问内存方面：指针是以地址作为值的变量，而数组名的值是一个特殊的固定地址，可以把它看作是常量指针，不能改变指针常量（数组名）的值
  • 函数定义中，被声明为数组的形参实际是一个指针。当传递数组时，按值调用传递其基地址，数组本身不被复制
  • 将数组名作为函数的实参，在被调用函数中，就能访问实参数组所在的存储单元，不但可以引用，还能改变这些单元的内容。返回主调函数后，相应数组元素的值就改变了

用指针实现内存动态分配

  1. 动态分配函数 void * malloc(unsigned size);: 在内存的动态存储区分配一连续空间，其长度为 size。若申请成功，则返回一个指向所分配内存空间的起始地址；若申请失败，则返回 NULL（值为 0）。该函数的返回值为 void * 类型，具体使用中，需要将 malloc 的返回值转换到特定指针类型，并赋给一个指针变量
  2. 动态存储释放函数 void free(void * ptr): 释放由动态存储分配函数申请到的整块内容空间，ptr 为指向要释放空间的首地址（不使用就要及时释放）

指针只有在被赋值以后才能被正确使用，即避免野指针
在 C 语言中，指针的算数运算只包括两个相同类型的指针相减以及指针加上或减去一个整数，其它操作都是非法的
