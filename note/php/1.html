三. 变量

3.1 & 3.2 定义变量

变量名通过 $ 标识，声明和使用时均需要带上 $ 符号
var_dump() 函数：输出变量数据类型，长度，值等

3.3 变量名命名规则

变量名必须以字母或下划线 _ 开头
变量名可包含字母，数字，下划线，和汉字
变量命名可使用下划线法或驼峰命名法

3.4 & 3.5 数据类型

memory_get_usage() 函数：返回当前 PHP 消耗的内存
数组：$array = array("", "");
PHP 支持 8 种原始类型，其中包括四种标量类型、两种复合类型和两种特殊类型
PHP 是一门松散类型的语言，不用声明变量的数据类型，PHP 会自动把变量转换为相应的数据类型

注：
根据操作系统、PHP 版本以及 PHP 的运行方式可能输出结果会不同。

3.6 ~ 3.12 标量类型

布尔类型

  • 只有两个值，TRUE & FALSE，不区分大小写
  • 使用 echo 指令输出布尔类型时，true 输出 1，false 什么也不输出
  • PHP 中一些量会自动转换为布尔值（可能在不同的 PHP 版本是不同的）

整型

  • 八进制和十六进制数分别在数字前加零 0 或 0x 即可

浮点型

  • 即小数，可用小数点也可用科学计数法表示，科学计数法可以使用 e 或 E

字符串

  • 字符串型的定义：单引号形式、双引号形式、Heredoc 结构形式
  • 字符串中的引号：在单引号中嵌入双引号；在双引号中嵌入单引号；使用转义符 \
  • 字符串中的 $：在单引号中会直接输出包含 $ 的字符串；在双引号中会替换为变量对应变量值
  • Heredoc：定界符 <<<，接着提供一个标识符（MYSELF），然后是长字符串，最后以此标识符结束字符串

注：
最后的标识符定要另起一行，并且此行除了标识符，并以 ; 结束之外，不能有任何其他字符，前后都不能有，包括空格，否则会出现错误。

3.13 & 3.14 特殊类型

资源

  • 由专门的函数来建立和使用的，例如打开文件、数据连接、图形画布
  • 对资源可以进行操作（创建、使用和释放）
  • 任何资源，在不需要的时候应该被及时释放。如果忘记释放资源，系统自动启用垃圾回收机制，在页面执行完毕后回收资源，以避免内存被消耗殆尽

注：
含中文的文本文件保存时要在 Notepad++ 选择 UTF-8 格式编码，否则会出现乱码。

空类型

  • NULL 是空类型，对大小写不敏感
  • NULL 类型只有一个取值，表示一个变量没有值
  • 当变量被赋值为 NULL，或者尚未被赋值，或者被 unset() 为 NULL
  • error_reporting(0); 禁止显示 PHP 警告提示
